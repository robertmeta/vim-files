INTRODUCTION                                *asyncrun*


This plugin takes the advantage of new apis in Vim 8 (and NeoVim) to enable
you to run shell commands in background and read output in the quickfix window
in realtime.


COMMANDS                                    *asyncrun-commands*


                                            *asyncrun-:AsyncRun*
:AsyncRun[!] [opts] {cmd} ...  Run shell cmd with parameters in background 
                               and output to quickfix. when "!" is included,
                               auto-scroll in quickfix will be disabled. 

                               Parameters are splited by space, if a 
                               parameter contains space, it should be quoted
                               or escaped as backslash + space (unix only).

                                            *asyncrun-:AsyncStop*
:AsyncStop[!]                  Stop the running job, when "!" is included, job
                               will be stopped by signal KILL.
